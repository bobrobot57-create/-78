# Подтверждение Cryptomus — код 54251872

Cryptomus требует подтвердить, что ты владелец бота. Добавь код в описание бота.

## Что сделать

1. Открой **@BotFather** в Telegram
2. Отправь `/mybots`
3. Выбери **клиентского бота** (тот, через который клиенты получают коды)
4. Нажми **Edit Bot** → **Edit Description**
5. Впиши в описание код: **54251872**
   - Можно временно заменить всё описание на этот код
   - Или добавить в конец: `54251872`
6. Сохрани
7. Вернись в **Cryptomus** → **VoiceLab project moderation**
8. Нажми кнопку **Check** (Проверить)
9. После успешной проверки — **удали код из описания бота** в BotFather

## Важно

После подтверждения Cryptomus **обязательно убери** код из описания — он нужен только для проверки.
